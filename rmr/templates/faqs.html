How to start using it?

To use RMR you need to add the books that you want to buy, using the "Add Book" link.
To add a new book, just put the book information, and don't foget to choose how much you desire the book.
When adding a new book, you can also add new informations about it, like a new author, genre, publisher and etc...
Then just head to "Book List" and find what you want there using the filters to help you finding what you want.
